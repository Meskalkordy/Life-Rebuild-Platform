### 📖 About the Project
**Life-Rebuild** هو منصة رقمية ذكية صُممت من قلب المعاناة في قطاع غزة، تهدف إلى أتمتة وتنظيم عملية التعافي للعائلات النازحة. نحن نربط الإدارة القانونية، توثيق الخسائر، والدعم النفسي في بيئة واحدة آمنة لتقليل البيروقراطية وتسريع الوصول للموارد الحيوية.

## 🚀 Key Features | ميزات النظام
* **📂 Digital Archive:** نظام سحابي مشفر لتوثيق الأوراق الثبوتية المفقودة.
* **🏠 Loss Reporting:** أداة مرنة لحصر الأضرار وتجهيز تقارير رسمية للمنظمات الدولية.
* **🗺️ Recovery Roadmap:** خطة عمل مخصصة (Step-by-Step) لكل عائلة.
* **🤝 NGO Linkage:** نظام ربط ذكي يضمن وصول المساعدات لمستحقيها بدون تكرار.
* **🧠 Psychosocial Hub:** مساحة دعم نفسي رقمي لتعزيز الصمود.

## 💡 How to Run the Project
1. **Clone the repo:** `git clone https://github.com/Meskalkordy/Life-Rebuild-Platform.git`
2. **Install dependencies:** `npm install`
3. **Configure Database:** Import the SQL file from `/database` folder.
4. **Run Server:** `npm start`
